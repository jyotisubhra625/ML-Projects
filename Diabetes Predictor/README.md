# Diabetes Predictor

This is a simple web application that predicts whether a person has diabetes based on a number of input features.

## Setup

1.  **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the application:**
    ```bash
    python app.py
    ```

3.  **Open your browser and go to:**
    [http://127.0.0.1:5000/](http://127.0.0.1:5000/)

## How to use

Fill in the form with the required information and click on the "Predict" button. The application will then display whether the person is likely to be diabetic or not.
